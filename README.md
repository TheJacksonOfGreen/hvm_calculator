# Calculator Project
For your first project this year, you are going to be creating a simple four-operation calculator.
- Your calculator must support addition, subtraction, multiplication, and division.
- Your calculator must correctly solve math problems it is given.
- Your calculator must support decimal and negative numbers.
- Your calculator must not crash if the user attempts to divide by zero. 

Example output #1 (User Input Italicized): 
> Enter an Operation: _add_\n
> Enter the First Number: _7_\n
> Enter the Second Number: _3_
> The Answer is 10.0

Example output #2: 
> Enter an Operation: _multiply_
> Enter the First Number: _25.2_
> Enter the Second Number: _31.4_
> The Answer is 791.28

Example output #3:
> Enter an Operation: _subtract_
> Enter the First Number: _-7_
> Enter the Second Number: _20_
> The Answer is -27.0

Example output #4:
> Enter an Operation: _divide_
> Enter the First Number: _82_
> Enter the Second Number: _0_
> Cannot divide by zero!
